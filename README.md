
# Yuvraj Pujary - Portfolio

Welcome to the **Portfolio** repository of Yuvraj Pujary, a Cloud Engineer specializing in AWS services. This project is a personal website showcasing my work, skills, certifications, and projects.

## 🚀 Live Demo
You can check out the live website [here](https://yuvrajpk.github.io/Portfolio/).

## 📝 Overview

This portfolio contains:

- **Home**: Introduction and overview of my work.
- **About**: More information about my background and skills.
- **Certificates**: A showcase of my professional certifications.
- **Projects**: A collection of my projects, with descriptions and links to repositories.
- **Resume**: A downloadable resume for easy access to my professional experience.
- **Contact**: Links to my social media profiles for easy communication.

## 💼 Technologies Used

This portfolio was built using:

- **HTML5**
- **CSS3**
- **GitHub Pages** for hosting

## 🌟 Features

- Fully responsive design.
- Clean and modern user interface.
- Integration with GitHub, LinkedIn, and other social platforms.
- Organized and easy-to-read sections for navigation.
  
## 📜 How to Use

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Yuvrajpk/Portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd Portfolio
   ```
3. **Open the `index.html` file** in your browser to view the website locally.
